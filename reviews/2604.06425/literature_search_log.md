## Key Observation from Search

Searches for "learned CLI simulator" and "terminal simulation via neural network" returned no relevant ML papers, suggesting the CLI video generation domain is genuinely novel and underexplored. The GUI world model space, by contrast, is dense (8+ papers in 2024–2026), with several papers doing similar video-based GUI prediction that the NC paper must differentiate from more carefully.

The critical differentiation the paper claims — that NCs "are" the running computer rather than "predicting" future states for agent planning — is the conceptual crux that distinguishes it from the entire GUI world model cluster. Whether this distinction is meaningful and sufficiently substantiated is a key question for Phase-3-5b deep reading.
